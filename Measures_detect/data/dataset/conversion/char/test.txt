全面推行证明告知承诺制实施方案	0
为贯彻落实党中央、国务院关于持续开展“减证便民”行动的决策部署，深化本市“放管服”改革、优化营商环境，根据《国务院办公厅关于全面推行证明事项和涉企经营许可事项告知承诺制的指导意见》(国办发〔2020〕42号)的精神，现就全面推行证明告知承诺制，制定以下实施方案	0
一、告知承诺实行范围	0
经市政府批准公布保留的本市机关和事业单位全部51项证明，除直接涉及国家安全、国家秘密、公共安全、金融业审慎监管、生态环境保护，直接关系人身健康、生命财产安全，以及重要涉外等风险较大、纠错成本较高、损害难以挽回的证明以外，逐步全面推行告知承诺制	2
同时，根据法律、法规或者国务院决定的立改废情况，对实行告知承诺制的证明在此基础上进行动态调整	2
二、告知承诺适用对象	0
申请人可自主选择是否采用告知承诺制方式办理	0
申请人不愿承诺或者无法承诺的，应当提交法律法规或者国务院决定要求的证明	0
申请人有较严重的不良信用记录或者存在曾作出虚假承诺等情形的，在信用修复前不适用告知承诺制	0
三、告知承诺证明办理流程	0
(一)证明索要单位告知	0
证明索要单位应当通过告知承诺书，向申请人告知下列内容	0
1.证明名称、证明内容、设定依据	0
2.准予办理应当具备的条件、标准、技术要求，以及不实承诺的法律后果等	0
3.所提供的服务内容、咨询方式，核查权力，采取的监管方式以及承担的责任等	0
4.认定申请人不实承诺的程序和标准，整改期限，违诺失信行为等级划分标准，以及申请人解释、说明、申诉、信用修复的渠道	0
5.承诺书是否公开、公开范围及时限	0
6.应当告知申请人的其他内容	0
(二)申请人承诺	0
申请人知晓告知内容，向证明索要单位确认和承诺下列内容	0
1.所填写的基本信息、提交的所需材料真实、合法、有效、完整	0
2.已经知晓告知的全部内容，且达到相应的条件、标准和技术要求	0
3.愿意承担不实承诺的法律责任，以及告知的违诺失信惩戒后果	0
4.所作承诺是申请人真实意思的表示	0
(三)接件与受理	0
证明索要单位收到承诺书后，对符合告知承诺条件的，应当当场采信，不再索要有关证明并依据书面承诺办理相关行政事项	0
对不符合告知承诺条件的，应当告知申请人改由一般程序申办	0
(四)核查与监管	0
证明索要单位要针对证明特点等分类确定核查办法，将承诺人的信用状况作为确定核查办法的重要因素，明确核查时间、标准、方式以及是否免予核查	2
证明开具单位要积极配合证明索要单位开展核查，提供有力的信息支撑	2
可免予核查的，要综合运用“双随机、一公开”监管、重点监管、“互联网+监管”、智慧监管等方式实施日常监管	2
对在核查或者日常监管中发现承诺不实的，要依法终止办理、责令限期整改、撤销行政决定或者予以行政处罚，并纳入信用记录	2
涉嫌犯罪的，依法移送司法机关	2
四、工作要求	0
(一)加强组织领导	1
各相关部门要高度重视推行证明告知承诺制工作，切实加强工作领导和组织实施	2
部门主要负责同志作为第一责任人，及时了解情况，研究解决问题	0

(三)加强研究细化	1

第一章　总　则	0
第一条　【定义内涵】本规定所称的对外委托鉴定评估工作(以下简称对外委托工作)，是指人民法院在审判和执行过程中委托专业机构或者专业人员进行鉴定、检验、评估、审计等工作，并进行监督协调的司法活动	0
第二条　【委托类别】基于审判和执行需要，人民法院可以就以下鉴定评估类别进行对外委托	0
(一)法医类、物证类、声像资料类、环境损害类由司法行政部门进行登记管理的鉴定类别	0
(二)动产、无形资产、企业价值、资产损失或者其他经济权益资产评估类别	0
(三)房地产评估类别	0
(四)建设工程造价、工程修复造价等工程造价鉴定类别	0
(五)会计审计鉴定类别	0
(六)知识产权鉴定类别	0
(七)其他需要鉴定评估的类别	0
第三条　【委托平台】充分运用信息技术，建设“北京法院对外委托一体化平台”(以下简称对外委托平台)，实现对外委托工作规范化、智能化、集约化、专业化、动态化管理	2
人民法院和专业机构应当通过对外委托平台开展工作，及时准确填报信息，依法规范做好司法公开工作	2
第四条　【名册制度】人民法院对外委托工作按照公开、公平、民主、择优的原则，实行对外委托专业机构名册制度	2
北京市高级人民法院负责编制《北京法院对外委托专业机构名册》(以下简称名册)	2
第五条　【职能部门、管理机制】对外委托辅助事务包括辅助审查、确定机构、办理委托、材料移转、勘验现场、参与听证、司法统计及沟通协调等工作	0
各院可以根据实际情况，合理确定本院诉讼服务部门集约开展对外委托工作辅助事务的范围	2
北京市高级人民法院诉讼服务部门负责全市法院对外委托工作的组织协调、监督指导和统一管理，利用对外委托平台构建包括专业机构业务能力、工作质效、服务水平等标准在内的动态管理和考核评价机制	2
第二章　委　托	0
第一节　确定机构	0
第六条　【鉴定评估提起】当事人可以就审判和执行过程中的专门性问题向人民法院申请鉴定评估	0
当事人未申请鉴定评估，人民法院对专门性问题认为需要鉴定评估，符合依职权调查收集证据条件的，应当委托具备资格的专业机构或者专业人员进行鉴定评估	2
准许当事人申请或者依职权委托的，法官应当确认需要鉴定评估的事项和要求	0
当事人申请的事项与待证事实无关联，或者对证明待证事实无意义的，法官应当不予准许	0
第七条　【信息告知】人民法院启动对外委托工作，应当告知当事人权利义务、风险责任及机构确定方式等内容，并做好记录	2
第八条　【机构确定方式】专业机构的确定以当事人协商选择与人民法院指定相结合的方式进行	0
当事人申请鉴定评估的，可以由双方当事人协商选择	0
当事人协商不成或者人民法院依职权委托鉴定评估的，由人民法院根据不同类别按照以下方式在名册内指定	0
(一)涉法医类、物证类、声像资料类、环境损害类鉴定的，由各院随机指定	0
(二)涉资产评估、房地产评估、工程造价鉴定、会计审计鉴定、知识产权鉴定类别的，由北京市高级人民法院诉讼服务部门随机指定	0
遇有特殊情况，经本院院长审核批准和纪检监察部门登记备案，报北京市高级人民法院诉讼服务部门同意，可以由各院随机指定	0
(三)其他鉴定评估类别，由各院随机指定或者自行指定	0
当事人协商不成或者人民法院依职权委托鉴定评估，有在名册内无法指定专业机构情形的，各院可以在名册外自行指定	0
当事人协商选择或者人民法院指定名册外专业机构的，法官应当对相关专业机构、专业人员的资格和专业能力进行审查，并决定是否委托	0
第九条　【征询】为提高确定专业机构的效率和准确性，缩短对外委托工作周期，案件委托鉴定评估后，专业机构因超出自身技术条件或者专业能力等情形决定不予受理或者终止鉴定评估的，人民法院可以通过对外委托平台向名册内相应类别项下的专业机构征询处理，并明确委托事项和要求，以确定专业机构的范围	2
涉法医类、物证类、声像资料类、环境损害类鉴定以及其他鉴定评估类别的，由各院诉讼服务部门审核征询内容，发布征询公告	0
涉资产评估、房地产评估、工程造价鉴定、会计审计鉴定、知识产权鉴定的，由北京市高级人民法院诉讼服务部门审核征询内容，发布征询公告	0
遇有特殊情况，经本院院长审核批准和纪检监察部门登记备案，报北京市高级人民法院诉讼服务部门同意，可以由各院诉讼服务部门审核征询内容，发布征询公告	2
经征询，有专业机构应询，属于当事人申请鉴定评估的，可以在应询的专业机构范围内重新进行协商选择	0
当事人协商不成或者人民法院依职权委托鉴定评估的，按照本规定第八条第一款规定的方式指定专业机构	2
无专业机构应询的，可以委托名册外的专业机构或者专业人员	0
第十条　【回避事由】专业机构或者专业人员有下列情形之一的，应当自行回避，当事人可以采用口头或者书面方式申请其回避	0
(一)曾经作为鉴定人、评估人员参加过同一个鉴定评估事项的	0
(二)曾经就同一个鉴定评估事项提供过专业咨询意见的	0
(三)曾经参与过同一个鉴定评估事项庭审质证的	0
(四)有法律、法规、司法解释等规定应当回避的	0
(五)其他可能影响独立、客观、公正开展工作的情形	0
当事人提出回避申请，应当说明理由	0
人民法院对当事人提出的回避申请应当予以审查	2
在确定专业机构时，人民法院发现专业机构或者专业人员有第一款规定情形的，应当回避相关专业机构或者专业人员	0
第十一条　【申请回避的处理】人民法院对当事人提出的回避申请，应当在申请提出的三日内作出决定	0
申请人对决定不服的，可以在接到决定时申请复议一次	0
复议期间，被申请回避的专业机构、专业人员，不停止参与本案的工作	0
人民法院对复议申请，应当在三日内作出复议决定，并通知复议申请人	0
当事人申请回避的理由成立，尚未办理委托的，终止办理委托	0
已经办理委托的，解除委托	0
申请回避的理由不成立的，驳回申请	0
第十二条　【重新确定机构】有当事人申请回避理由成立、专业机构不予受理或者终止鉴定评估、人民法院解除委托或者决定重新鉴定评估等情形，需要重新确定专业机构的，人民法院应当按照本规定重新确定专业机构	2

第三节　委托费用	0

第二十一条　【费用退还】专业机构终止鉴定评估工作、人民法院解除委托，或者专业人员不履行出庭义务致使鉴定意见、评估报告不被采纳的，专业机构应当退还相应费用	0

第二十六条　【鉴定评估原则】专业机构、专业人员应当依法独立、客观、公正地开展鉴定评估工作	0

(四)法律、法规、司法解释等规定的其他情形	0

(二)当事人拒不配合或者鉴定评估工作受到严重干扰，致使鉴定评估工作无法继续进行的	0
(三)当事人拒绝支付委托费用的	0
(四)因不可抗力等因素致使鉴定评估工作无法进行的	0
(五)符合应当回避情形的	0
(六)其他法律、法规、司法解释等规定可以终止鉴定评估工作的情形	0

第一条【要素式审判的含义】　要素式审判方法，主要是对固定案情的基本事实要素进行提炼，就各要素是否存在争议进行归纳，并围绕争议要素进行庭审及制作裁判文书，简化审理流程，提高审判效率，实现简案快审	0
要素式裁判文书是指对于某些能够概括出固定要素的案件，在撰写裁判文书时不再按照传统的裁判文书格式分开陈述原告诉称、被告辩称、本院查明和本院认为部分，而是围绕着具体的案件要素，陈述原、被告意见及列明证据和法院认定理由及依据的法律文书	0
第二条【适用要素式审判的案件类型】　各法院应将以下七类案件直接导入多元调解和速裁程序处理，多元调解不成但事实清楚、权利义务关系明确、当事人争议不大的普通民商事案件适用要素式审判方式审理	0
(一)诉讼标的额在50万元以下的买卖合同纠纷	0
(二)诉讼标的额在50万元以下的民间借贷纠纷	0
(三)信用卡纠纷	0
(四)金融借款纠纷	0
(五)追索物业费、供暖费的物业服务合同纠纷、供用热力合同纠纷	0
(六)机动车交通事故责任纠纷	0
(七)家事纠纷，包括婚姻家庭、继承、赡养、抚养、扶养等纠纷	0
在立案庭接收上述案件时，应对起诉状、证据进行扫描存档	0
第三条【要素表的填写】　根据要素式案件的特点确定事实要素	0
在立案阶段，由立案法官指导原告填写案件要素表	0
